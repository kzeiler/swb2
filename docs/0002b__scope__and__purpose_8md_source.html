1 Scope and Purpose {#scope}
2 ===========================
3 
4 This report documents the Soil-Water Balance (SWB) computer code,
5 which
6 is designed to calculate the spatial distribution of groundwater
7 recharge over time using a gridded data structure. The SWB model
8 should
9 yield better results than can be obtained by assuming that a fraction
10 of
11 precipitation converts to recharge; conversely, the SWB model is much
12 simpler and less time-intensive to apply than a fully coupled
13 groundwater and surface-water model
14 (Jyrkama and others, 2002; Markstrom and others, 2008).
15 
16 The theoretical basis, data requirements for use, and limitations and
17 assumptions relating to the SWB code are presented in this report. In
18 addition, two test cases are provided. The first test case (Maui,
19 Hawaii) confirms the performance of the SWB code relative to the
20 Hawaii
21 Water Budget code. The second test case demonstrates the application
22 of
23 the SWB code to the Little Plover River in Wisconsin.
24 
25 Appendixes to this document contain a complete listing of control file
26 directives and a description of input and output files use with SWB.
