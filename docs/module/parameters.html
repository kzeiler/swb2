PARAMETERS_T

module to provide a single POC for storage and retrieval of parameter scalars and vectors need to provide a method for storing, finding, and retrieving parameters. dictionary: keyword, string_list

first create list of files. parse through each file, adding to the param dictionary. once complete, allow other modules to interrogate the dictionary. return matches in the data type required for the parameter. once all params are in place, the data structures can be deallocated.
